# cpp_hp_Dariga_Tolebay

1. В чём отличие динамического массива от статического массива в языке C++?

Статический массив имеет фиксированный размер, который задаётся при компиляции, и память для него выделяется автоматически.
Динамический массив создаётся во время выполнения программы, его размер можно задать в процессе работы, память выделяется вручную с помощью new и должна освобождаться через delete[].
